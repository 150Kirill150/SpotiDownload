# SpotiDownload
### SpotiDownload - Скачивание текущего трека в 1 клик! ###



+ Системные требования
  - OS: Linux

+ Как устанавливать?
  - <pip install pyrogram tg-crypto>
  - Зайти в телеграмм
    - Через поиск найти бота @spotify_down_bot
    - Нажать старт
    - Перейти по ссылке: [id] https://t.me/spotify_bot_updates
    - Подписатся на канал
